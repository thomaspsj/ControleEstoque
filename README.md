# ControleEstoque

## 📋 Apresentação da Aplicação

**ControleEstoque** é uma aplicação desenvolvida para facilitar a gestão de estoques, fornecendo funcionalidades completas para cadastro, operação e análise de dados.

## 🚀 Funcionalidades

### 🌐 Infraestrutura
- Tratamento de exceção
- Log
- Autenticação
- Autorização
- Auditoria

### 📁 Cadastros
- Grupos de Produtos
- Marcas de Produtos
- Locais de Armazenamento
- Unidade de Medida
- Países
- Estados
- Cidades
- Fornecedores
- Produtos
- Perfis de Usuários
- Usuários

### 🔄 Operações
- Entrada de produtos
- Saída de produtos
- Lançamento de perdas
- Inventário

### 📊 Relatórios
- Posição de Estoque
- Ressuprimento

### 📈 Gráficos
- Perdas por período
- Entradas vs Saídas por período
